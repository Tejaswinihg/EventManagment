# Event Management System

A simple full-stack web application to manage event registrations and details.  
This project demonstrates frontend design, backend form handling, and database integration for a real-world use case.

---

##  Features

- Event listing pages with detailed event information  
- User registration for events through HTML forms  
- Backend form handling using PHP  
- Data storage and retrieval using MySQL  
- Basic validation and structured UI flow  

---

##  Tech Stack

- **Frontend:** HTML, CSS  
- **Backend:** PHP  
- **Database:** MySQL  
- **Tools:** XAMPP / WAMP, VS Code  

---


---

##  How to Run Locally

1. Install **XAMPP** or **WAMP**
2. Copy the project folder into `htdocs`
3. Start **Apache** and **MySQL**
4. Create the database and update credentials in `db_connection.php`
5. Open in browser: http://localhost/EventManagement/index.html
